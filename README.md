## 🚀 我的 AI 项目实践

### 📈 个股智能分析系统（AI Agent + RAG）

> 基于 Eino + Redis Stack 构建的投研助手，实现自然语言→结构化查询→多源数据检索→生成报告

- ✅ **意图理解层**：实体知识库（股票/指标/时间）+ 槽位组合规则 + 示例库检索，复杂查询准确率 86%
